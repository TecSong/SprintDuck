# 高级 AI 应用后端工程师 JD

> 本文档为测试用虚构 JD，所有公司、岗位与业务描述均为模拟数据。

## 公司介绍

星途智能是一家面向企业客户提供 AI 工作流平台的 SaaS 公司，核心产品包括企业知识库问答、智能客服、Agent 自动化任务编排、销售运营助手和内部研发提效工具。

我们正在建设下一代企业级 AI Agent 平台，希望将大模型能力、业务系统、权限体系和可观测工程能力结合起来，帮助客户把复杂流程自动化。

## 岗位名称

**高级 AI 应用后端工程师**

## 工作地点

- 上海 / 杭州 / 北京
- 支持混合办公
- 优秀候选人可远程

## 岗位职责

- 负责 AI Agent 平台后端服务的设计、开发和维护，包括任务编排、工具调用、用户确认、权限控制和执行状态管理。
- 参与企业知识库问答系统建设，优化文档解析、分块、检索、重排、引用溯源和答案生成链路。
- 设计稳定可靠的异步任务系统，支持长任务执行、失败重试、超时控制、取消任务和任务恢复。
- 与前端、算法、产品和交付团队协作，将客户场景抽象为可复用的平台能力。
- 建设评测与回归体系，持续跟踪模型效果、响应延迟、调用成本、失败率和用户满意度。
- 完善线上服务的日志、指标、链路追踪和告警，提高系统可观测性与问题定位效率。
- 参与核心服务的架构演进，提升系统的扩展性、稳定性、安全性和可维护性。

## 任职要求

### 必备条件

- 5 年以上后端开发经验，有复杂业务系统或平台型产品建设经验。
- 熟悉 Python，能够熟练使用 FastAPI、Pydantic、SQLAlchemy、pytest 等常见工程工具。
- 熟悉 PostgreSQL、Redis、消息队列和常见缓存策略，理解事务、索引、锁和性能优化。
- 具备良好的 API 设计能力，理解 REST、Webhook、OpenAPI、鉴权、限流和幂等设计。
- 有生产环境问题排查经验，能通过日志、指标和链路追踪定位复杂问题。
- 具备良好的工程习惯，重视测试、代码可读性、接口契约和文档沉淀。

### 加分条件

- 有 LLM 应用、RAG、Agent、工具调用或 Prompt 编排相关项目经验。
- 熟悉 OpenAI API、LangChain、LangGraph、LangSmith、向量数据库或 pgvector。
- 有多租户 SaaS、权限系统、审计日志、工作流引擎或低代码平台经验。
- 有 ClickHouse、Elasticsearch、Kafka、Kubernetes 或云原生部署经验。
- 有开源项目、技术博客、技术分享或团队工程规范建设经验。

## 团队协作方式

- 小团队敏捷迭代，每 2 周一个开发周期。
- 工程师会参与需求澄清、技术方案设计、上线验证和复盘。
- 代码通过 Pull Request 合并，要求包含必要测试和清晰的变更说明。
- 重大技术方案需编写简短设计文档，说明目标、边界、风险和回滚方式。

## 你将参与的项目

### Agent 执行引擎

构建一个可观测、可恢复、可扩展的 Agent 执行引擎，支持模型推理、工具调用、人工确认、任务暂停、失败重试和事件流输出。

### 企业知识库问答

优化从文档入库到答案生成的完整链路，包括 Markdown、PDF、Word、网页等多种格式解析，以及引用溯源和权限过滤。

### AI 应用评测平台

建设面向研发和交付团队的评测工具，支持测试集管理、批量运行、结果对比、失败样本归因和模型配置实验。
